# HNMU 科研教学智能工作台

欢迎使用科研教学多智能体工作台。领导 Agent 会调度执行团队，并审核每一项交付是否达标。

## 工作模式

- **科研协作**：文献检索、综述、方法与创新点、实验设计、论文初稿
- **教学备课**：教学目标、课堂活动、案例、评价量规与课后任务
- **文献精读**：研究问题、方法判断、证据等级、局限性与课堂讨论题

## 质量机制

所有执行 Agent 的产出都会进入领导 Agent 审核。领导 Agent 将给出质量评分、审核意见，并对未达标内容直接生成修订后的最终交付。

当前可使用**全本地模式**。配置 DeepSeek key 并将 `.env` 中 `ALL_LOCAL=false` 后，将启用混合部署；实验设计和领导审核仍走本地模型。
